Remote Control Robot Vacuum

Overview

This robot is a remote-controlled wheeled platform built from a Shark RV1001AE vacuum that was discarded because of broken parts. I salvaged the vacuum and gutted the internal electronics, replacing them with a custom control system. These custom electronics consisted of an ESP32, Cytron MDD3A motor controller, and a LiPo battery.

I wrote a Python script that reads data over USB from an Xbox controller, then uploads that data to the ESP32's WiFi Access Point. The ESP32 then uses that controller input to drive the robot's wheels through the motor controller. This robotic platform will serve as a basis for exploring wheeled autonomy like odometry and pure pursuit.
